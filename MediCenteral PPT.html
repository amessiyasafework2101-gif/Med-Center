PharmaCare

Secure Pharmacy Management Platform

  • Multi-role SaaS for patients, pharmacies & administrators
  • Vanilla HTML, CSS & JavaScript
  • Firebase + Cloudinary + AI assistant (Dr. Aklilu)

The Problem

Why PharmaCare?

  • Patients need a trusted way to submit doctor results and order medicines safely
  • Pharmacies must verify licenses, staff CVs, and authentic prescriptions
  • Paper workflows are slow, error-prone, and difficult to audit
  • PharmaCare digitizes registration, approval, verification, and payments

Four User Roles

Role-based secure portals

  • Patient — submit results, view recommendations, order medicines
  • Pharmacy Owner — register pharmacy, approve pharmacist CVs, inventory
  • Pharmacist — verify doctor papers, payments, point-of-sale (New Sale)
  • Admin — approve pharmacy licenses, oversee platform data

Registration & Approval Flow

Trust at every step

  • Pharmacy owner registers → uploads license → admin approves
  • Pharmacist applies to selected pharmacy → owner approves CV
  • Patient submits doctor result paper to an approved pharmacy
  • Pharmacist verifies authentic result → patient can order medicines

Key Features

Built for real pharmacy operations

  • Dr. Aklilu — AI assistant for medicine doses, pain & symptoms
  • Payment proof — external pay + screenshot/ID → pharmacist approval
  • Pharmacy bill — issued to patient after order approval
  • Inventory, POS, orders dashboard, dark mode, responsive UI

Technology Stack

Modern lightweight architecture

  • Frontend: HTML5, Tailwind CSS, Lucide icons, Chart.js
  • Auth & database: Firebase Authentication + Firestore
  • Files: Cloudinary (licenses, CVs, result images, payment screenshots)
  • State sync: localStorage + real-time Firestore; secure role-matched login

Summary & Uses

PharmaCare in practice

  • Digital onboarding for licensed pharmacies and pharmacists
  • Safe patient ordering only after verified doctor results
  • Admin governance of pharmacy quality on the platform
  • Foundation for telepharmacy, analytics & mobile expansion
